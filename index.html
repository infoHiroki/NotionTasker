NotionTasker

ChromeブラウザからNotionにタスクを素早く追加できる拡張機能

主な機能

  • テキスト入力からNotionのデータベースにタスクを追加
  • シンプルで使いやすいUI
  • ショートカットキーでの素早い操作
  • Notionアカウントとの安全な連携

使用方法

  1. Chrome Web Storeから拡張機能をインストール
  2. 拡張機能の設定画面でNotionのAPIキーとデータベースIDを設定
  3. Chrome拡張機能のアイコンをクリックしてポップアップを開く
  4. タスクを入力し、送信ボタンをクリックまたはCtrl+Enterでタスクを追加

ショートカット

  • Alt+N: NotionTaskerのポップアップを開く
  • Ctrl+Enter: タスク入力欄でタスクを送信

セキュリティ

  • ユーザーのNotion APIキーとデータベースIDは、ローカルのChrome StorageにAESによって暗号化されて保存
  • タスク内容は、Notionのサーバーに直接送信され、拡張機能のサーバーには保存されない
