NotionTasker

NotionにタスクをすばやくKに追加するChrome拡張機能

機能

  • ChromeブラウザからNotionにタスクを簡単に追加
  • シンプルで使いやすいUI
  • Notionデータベースとのシームレスな連携

使用方法

  1. Chrome Web Storeから拡張機能をインストール
  2. NotionのAPIキーとデータベースIDを設定
  3. Chrome拡張機能のポップアップからタスクを入力して追加

インストール

Chrome Web Storeでインストール
